# 🛠️ 투두리스트 프로젝트
## 프로젝트 설명
## TypeScript로 간단한 투두리스트 애플리케이션을 제작했습니다. 주요 기능은 다음과 같습니다:

- 할 일 추가: 새로운 할 일을 추가
- 할 일 삭제: 완료된 할 일 삭제
- 할 일 상태 변경: 진행 중 ↔ 완료 상태 전환
